Dream Data Lakehouse

A scalable AWS lakehouse and Bedrock-based pipeline for ingesting, storing, and analyzing dream and film data through structured ETL and ML workflows. Leverages FastAPI, S3 lakehouse architecture, Claude for text generation, and Luma Ray 2 for asynchronous video generation.

Key Features

  • Bedrock Integration: Leverages Claude for synchronous text generation (movie scripts, analysis) and Luma Ray 2 for asynchronous video generation
  • S3 Lakehouse: Raw event logs stored in partitioned JSON format with append-only architecture
  • ETL Pipeline: AWS Glue transforms raw data into curated Parquet format for efficient querying
  • Analytics: Amazon Athena enables SQL queries on dream patterns, moods, motifs, and outcomes
  • Async Video Processing: Poll-based status checking for Luma Ray 2 video generation with S3 output delivery
  • FastAPI Backend: RESTful API for dream journal entries, render job management, and status polling
  • Theater UI: Immersive curtain-loading experience with autoplay video and calendar-based dream log
